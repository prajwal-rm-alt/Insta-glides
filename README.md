# Insta Glide

Insta Glide is a seamless, high-quality media downloader for Instagram. It allows you to instantly download Reels, Posts, and Carousels in their original high quality with a clean, modern interface.

## Features

- **Instant Downloads**: Just paste the URL and download high-quality videos or images directly.
- **Batch Processing**: Need to download multiple items? Queue them up in the Batch tab and download them all at once.
- **Saved History**: Easily keep track of your downloaded items in your local history so you can retrieve them later without going back to Instagram.
- **HD Quality**: Retrieves the highest resolution available directly from the source.
- **Sleek Interface**: Built with modern web technologies, providing a buttery-smooth and responsive experience.

## Tech Stack

- **Frontend**: React 19, TypeScript, Tailwind CSS, Motion (framer-motion)
- **Backend/API**: Node.js, Express
- **Downloader Core**: snapsave-media-downloader

## Getting Started

1. Clone this repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
